# Anushka Kandhway — Portfolio

Personal portfolio website built with HTML, CSS & JavaScript. Features a purple-themed dark design with custom cursor, GitHub API integration, contact form, and scroll animations.

## Sections
- Hero (B.Tech Student · Builder · Creator)
- About (interests, stats, exploring fields)
- Skills (10 skill cards with icons)
- Projects (Tourist Junction featured)
- GitHub Activity (live API data + contribution chart)
- Contact (links + message form)

## Live Site
Deployed via GitHub Pages: https://kandhwayanushka-hue.github.io/60-DAYS-TO-AI

## Tech
- Vanilla HTML/CSS/JS
- GitHub REST API
- ghchart.rshah.org for contribution graph
